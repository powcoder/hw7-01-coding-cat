* Texas Instruments TWL4030/6030 RTC

Required properties:
- compatible : Should be "ti,twl4030-rtc"
- interrupts : Should be the interrupt number.

Example:
	rtc {
		compatible = "ti,twl4030-rtc";
		interrupts = <11>;
	};
